### 👍Code example

- import `import org.furkankayam.TCValidation;`

```java
Boolean tcValid = TCValidation.tcValidation(
        "12345678901", // Turkish ID Number
        "mehmet furkan", // First Name
        "kaya", // Last Name
        "2001"); // Year of Birth

System.out.println(tcValid); // true-false
```
